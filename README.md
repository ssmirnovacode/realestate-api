# REST API for MYrealestate web application

API accessible on heroku:
https://myrealestate-api.herokuapp.com/properties

## Technologies and libraries used
 - Node.js
 - Express
 - MongoDB
 - Mongoose

### Endpoints:
 - GET /properties/sale
 - GET /properties/rent
 - GET /properties/id
 - GET /feedback
 - POST /feedback
 - POST /send-request
 - POST /contact
