# REST API for MYrealestate web application

https://ssmirnovacode.github.io/my-realestate/

API accessible on heroku:
https://realestate-rest.herokuapp.com/

## Technologies and libraries used
 - Node.js
 - Express
 - MongoDB
 - Mongoose

### Endpoints:
 - GET /properties/sale
 - GET /properties/rent
 - GET /properties/id
 - GET /feedback
 - POST /feedback
 - POST /send-request
 - POST /contact
